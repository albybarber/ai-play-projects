# Bea's Kitten Games

A collection of browser games featuring Bea's cat photos, with pink sparkle animations throughout.

**[Play now](https://albybarber.github.io/ai-play-projects/beaskittengames/)**

## Games

- **Memory Match** — Flip cards to find matching kitten pairs. Supports 1 and 2 players with 3 difficulty levels.
- **Kitten Catcher** — Whack-a-mole style game. Click kittens before they hide in a 30-second round.
- **Sliding Puzzle** — Slide tiles to reassemble a kitten photo. 3x3, 4x4, or 5x5 grids.

## Tech

Pure HTML, CSS, and JavaScript. No dependencies, no build step. Just open `index.html` in a browser.
