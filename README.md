# ai-play-projects

A collection of small utility scripts and experiments.

## Projects

- [webhook-action-string](./webhook-action-string) — Convert strings into webhook-style action strings (e.g. `file.upload.completed`)
- [aip-calculator](https://albybarber.github.io/ai-play-projects/aip-calculator/aip-calculator.html) — Browser-based Annual Incentive Plan bonus calculator with proration
- [beaskittengames](https://albybarber.github.io/ai-play-projects/beaskittengames/) — Bea's Kittens: 3 browser games (Memory Match, Kitten Catcher, Sliding Puzzle) featuring personal cat photos
- [make-a-thing](https://albybarber.github.io/ai-play-projects/make-a-thing/) — LEGO building & guessing companion app: category dice, word selection, scoring & turn management for 2-8 players
