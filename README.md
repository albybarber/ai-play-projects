# ai-play-projects

A collection of small utility scripts and experiments.

## Projects

- [webhook-action-string](./webhook-action-string) — Convert strings into webhook-style action strings (e.g. `file.upload.completed`)
- [aip-calculator](https://albybarber.github.io/ai-play-projects/aip-calculator/aip-calculator.html) — Browser-based Annual Incentive Plan bonus calculator with proration

> **Note:** When adding new HTML projects, enable a live demo via GitHub Pages where possible. The site is served from the `main` branch root at `https://albybarber.github.io/ai-play-projects/`.
